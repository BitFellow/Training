# Training Repository

This repository contains various projects and exercises related to data science, machine learning, and sustainability. It serves as a learning resource for individuals interested in these fields.

## Projects
- **K-means Clustering**: A project that demonstrates the use of K-means clustering algorithm on environmental factors data. The project includes data preprocessing, model training, and evaluation using silhouette score.

## Getting Started

To get started with the projects in this repository, follow these steps:
1. Clone the repository to your local machine.

2. Install the required dependencies using pip.

3. Navigate to the project directory and run the Jupyter Notebook to explore the code and results
